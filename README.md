🏟️ Projeto de Exemplo - Futebol Clube

Olá! Eu sou Guilherme Ramon e este é um projeto de exemplo de uma Atividade Extensionista realizada durante o primeiro semestre do curso de Análise e Desenvolvimento de Sistemas.

⚠️ Observação: este repositório é apenas um projeto de teste/exemplo. O site real encontra-se em outro repositório.

🎯 Objetivo do Projeto

O principal objetivo foi criar um site para um time de base chamado Base do Potiguar de Melancias, permitindo:

Apresentar o time e seus patrocinadores;

Exibir o calendário de treinos e a localização dos campos;

Proporcionar uma navegação simples, intuitiva e agradável.
